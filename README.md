## ISR-FABEL Dataset

This repository refers to the **ISR-FABEL dataset**, a multimodal dataset for children’s emotion recognition based on child–robot interaction sessions, including facial and body landmark annotations.

The dataset is described in:

> B. Ribeiro, A. Lopes, C. Carona, and U. J. Nunes,  
> *ISR-FABEL: A Landmark-Based Dataset and Multimodal Emotion Recognition Framework for Child-Robot Interaction*,  
> IEEE Access, 2025.  
> https://ieeexplore.ieee.org/document/11153483

### Access

Due to ethical and privacy considerations, the dataset is **not publicly available**.  
Access is granted for **non-commercial research purposes only** upon request.

Researchers interested in accessing the dataset must complete and sign the dataset user agreement provided in this repository. Approved users will receive access instructions by email.
